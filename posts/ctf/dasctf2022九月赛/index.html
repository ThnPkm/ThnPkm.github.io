dino3d

一个前端小游戏

抓包 看到/check.php 和 post 的参数，是score我们要够1000000，后面的checkCode和tm应该也要构造

翻js源码，美化一下

找sn() ，发现e就是source，t=checkCode ，source随便我们写

找checkCode ，DASxCBCTF_wElc03e

可以构造了，写个脚本

import time
import hashlib
import requests

url = "http://node4.buuoj.cn:28130/check.php"

score = '1000000'
code= 'DASxCBCTF_wElc03e'
checkCode = hashlib.md5((score+code).encode()).hexdigest()
tm = int(str(time.time())[0:10]) #这里最终要为int,否则脚本跑的不好

data = {
    'score' : score,
    'checkCode' : checkCode,
    'tm' : tm
}
print(checkCode)
print(tm)
res = requests.post(url=url,data=data)
print(res.text)
