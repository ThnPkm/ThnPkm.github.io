构造一下pop链：

Tset类 __destruct进入，有echo可以触发User类 __toString

 function __toString(){
            $cont = $this->fname;
            $size = $this->fsize;
            echo $cont->$size;

        }

重点是这里怎么走，这里有个逻辑$cont->$size 令$cont为Show类 则可以触发Show类__get

__get():当我们试图获取一个不可达属性时(比如private)或不存在，则类会自动调用__get函数
__call():着调用一个对象中不存在或被权限控制中的方法，__call 方法将会被自动调用。

这里的$size也是我们可以控制的，Show类里找不到$size这个属性，就会触发__get

 function __get($name)
        {
            $this->ok($name);
        }

这个__get() 还是有参数的$name, 搞得挺烦人，但是$name其实就是刚调用的不可达属性$size

继续走， $this->ok($name);调用了不可达的方法ok()，所以触发了__call，来看看__call

public function __call($name, $arguments)
        {
            if(end($arguments)=='phpinfo'){
                phpinfo();
            }else{
                $this->backdoor(end($arguments));
            }
            return $name;
        }

也是好烦，里面怎么有两个参数($name, $arguments)，解释一下。

这里的$name就是刚调用的不可达的方法也就是ok(),跟__get($name)里的$name不一样。

这里的$arguments就是ok()自带的参数也就是ok($name)里面的$name,即我们可控的$size

我认为这里还挺绕，刚开始没理解 ，这样解释一下就好很多

回到起点，我们可以控制的$size应该附什么值？

$this->backdoor(end($arguments));

我们pop链子最终的目的就是调用backdoor(),$arguements应该是我们要包含的文件，所以$size就是我们要包含的文件

function getQueryVariable(variable)
{
       var query = window.location.search.substring(1);
       var vars = query.split("&");
       for (var i=0;i<vars.length;i++) {
               var pair = vars[i].split("=");
               if(pair[0] == variable){return pair[1];}
       }
       return(false);
}

function check(){
