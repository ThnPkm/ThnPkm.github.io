java反射概念

任意一个类，都能够获取到这个类的所有属性和方法，对于任意一个对象，都能够调用它的任意一个方法和属性(包括私有的方法和属性)通过“反射”，我们可以将Java这种静态语⾔附加上动态特性。

所谓“动态特性”，p神有精辟的总结：⼀段代码，改变其中的变量，将会导致这段代码产⽣功能性的变化，我称之为动态特性

java反射机制给漏洞利用提供了很多便利。所以，学习java安全是绕不开它的。

反射使用

获取类

  • forName()方法 需要知道类名 比如"java.lang.Runtime"
  • 直接引用类的class属性 其实不属于反射方式
  • getClass()方法 上下⽂中存在某个类的实例 对象 ，那么我们可以直接通过 对象.getClass() 来获取它的类
package java反射;

public class Test1 {
    public static void main(String[] args) throws ClassNotFoundException {

        //1.forName()方法 需要知道类名 比如"java.lang.Runtime"
        Class name1 = Class.forName("java.lang.Runtime");
        System.out.println(name1); //class java.lang.Runtime

        //2.直接引用类的class属性 其实不属于反射方式
        Class name2 = Runtime.class;
        System.out.println(name2); //class java.lang.Runtime

        //3.getClass()方法 上下⽂中存在某个类的实例 对象 ，那么我们可以直接通过 对象.getClass() 来获取它的类
        Runtime rt = Runtime.getRuntime();
        Class name3 = rt.getClass();
        System.out.println(name3);  //class java.lang.Runtime

    }
}

获取类方法

Class类提供了以下几个方法来获取Method：

  • Method[] getDeclaredMethods()：获取当前类的所有Method（不包括父类）

包括public、protected、private和默认方法

import java.lang.reflect.Method;

public class Test2 {
    public static void main(String[] args) throws ClassNotFoundException {
        Class  name1 = Class.forName("java.lang.Runtime");
        System.out.println(name1);
        Method[] m = name1.getDeclaredMethods();
        for(Method x:m)
            System.out.println(x);
    }
}
  • Method getDeclaredMethod(name, Class...)：获取当前类的某个Method（不包括父类）

获取特定的方法，第一个参数是方法名，第二个参数是该方法的参数对应的class对象

例如这里Runtime的exec方法参数为一个String，所以这里的第二个参数是String.class

import java.lang.reflect.Method;

public class Test2 {
    public static void main(String[] args) throws ClassNotFoundException , NoSuchMethodException{
        Class  name = Class.forName("java.lang.Runtime");
        Method m = name.getDeclaredMethod("exec",String.class);
        System.out.println(m);
        //public java.lang.Process java.lang.Runtime.exec(java.lang.String) throws java.io.IOException
    }
}
  • Method getMethod(name, Class...)：获取某个public的Method（包括父类）
  • Method[] getMethods()：获取所有public的Method（包括父类）

获取构造方法

通过Class实例获取Constructor的方法如下：

  • getConstructor(Class...)：获取某个public的Constructor；
  • getDeclaredConstructor(Class...)：获取某个Constructor；
  • getConstructors()：获取所有public的Constructor；
  • getDeclaredConstructors()：获取所有Constructor。

调用非public的Constructor时，必须首先通过setAccessible(true)设置允许访问。setAccessible(true)可能会失败。

获取类成员变量

  • getDeclaredFields() 能够获得类的成员变量数组，包括public、private、proteced。不包括父类
  • getDeclaredField(String name) 获取特定的，参数是想要的方法的名称
  • getFields() 同理，只能获得public的，但是包括了父类的
  • getField(String name) 同理，参数是想要的方法的名称

如下例子

import java.lang.reflect.Constructor;
import java.lang.reflect.Field;
import java.lang.reflect.Method;


class Student{
     private String name;
     public int age;

     public Student(){
         name = "ThnPkm";
         age = 20;
     }

     private Student(String name1){
         name= name1;
     }



}

public class Test3 {
    public static void main(String[] args) {

        Student student = new Student();
        //获取类对象
        Class name = student.getClass();

        // 获取构造函数
        Constructor[] constructors = name.getDeclaredConstructors();
        for (Constructor m:constructors)
            System.out.println(m); 
        
        //获取类成员变量
        Field[] a = name.getDeclaredFields();
        for (Field m:a)
            System.out.println(m);
    }
}

反射创建类对象

  • newInstance() 反射来生成实例化对象
Class c = Class.forName("com.reflect.MethodTest"); // 创建Class对象
Object m1 =  c.newInstance(); // 创建类对象

使用Class对象的newInstance()方法来进行创建类对象

只需要通过forName方法获取到的class对象中进行newInstance方法创建即可

  • invoke()

invoke方法位于java.lang.reflect.Method类中，用于执行某个的对象的目标方法,一般会和getMethod方法配合进行调用。

public Object invoke(Object obj, Object... args)

第一个参数为类的实例，第二个参数为相应函数中的参数

先forName拿到Class，再newInstance获取类对象，再getMethod获取方法，然后调用

方法.invoke(类或类对象)

例子：

package java反射;

import java.lang.reflect.InvocationTargetException;
import java.lang.reflect.Method;

public class Test4 {
    public void test(){
        System.out.println("测试成功");
    }
    
    public static void main(String[] args) throws ClassNotFoundException, InstantiationException, IllegalAccessException, InvocationTargetException, NoSuchMethodException {
        Class c = Class.forName("java反射.Test4");
        System.out.println(c); //获取Class 类

        Object o = c.newInstance();
        System.out.println(o); //获取类对象
        
        Method m = c.getMethod("test");
        System.out.println(m); //获取方法
        m.invoke(o); //使用invoke 调用到test()
    }
    
}

由上可见 利用java反射的机制，我们可调用任何类的变量 方法 修改变量值等等。 通过某些技巧是可以无视private protected修饰的。

如何利用这种方式来实现rce呢，下面看下例子

RCE示例

Runtime类里面有一个exec方法，可以执行命令

import java.lang.reflect.Constructor;
import java.lang.reflect.InvocationTargetException;
import java.lang.reflect.Method;

public class Exec {
    public static void main(String[] args) throws ClassNotFoundException, NoSuchMethodException, InvocationTargetException, IllegalAccessException, InstantiationException {
        Class c = Class.forName("java.lang.Runtime");
        Constructor con = c.getDeclaredConstructor();
        con.setAccessible(true);
        Method m = c.getMethod("exec",String.class);
        m.invoke(con.newInstance(),"calc");
    }
}

1.用forName拿到类

2.用getDeclaredConstructor()拿到构造方法

3.setAccessible(true) 来突破Runtime 类的构造方法是private的限制 （这里我没成功）

4.getMethod(“exec”,String.class) 拿到exec方法

5.con.newInstance() 来实例化类

6.m.invoke(con.newInstance(),“calc”); 用invoke来调用

换个方式 ,通过 Runtime.getRuntime() 来获取到 Runtime 对象，不去调用构造方法了

public class Exec {
    public static void main(String[] args) throws Exception {
        Class clazz = Class.forName("java.lang.Runtime");
        Method execMethod = clazz.getMethod("exec", String.class);
        Method getRuntimeMethod = clazz.getMethod("getRuntime");
        Object runtime = getRuntimeMethod.invoke(clazz);
        execMethod.invoke(runtime, "calc.exe");
    }
}

如果一个类没有无参构造方法，也没有类似单例模式里的静态方法，我们怎样通过反射实例化该类呢？

getConstructor 接收的参数是构造函数列表类型，因为构造函数也支持重载，所以必须用参数列表类型才能唯一确定一个构造函数。

常用的另一种执行命令的方式ProcessBuilder，我们使用反射来获取其构造函数，然后调用start() 来执行命令：

public class Exec {
    public static void main(String[] args) throws Exception {
        Class clazz = Class.forName("java.lang.ProcessBuilder");
        Constructor con = clazz.getConstructor(List.class);
        Object o = con.newInstance(Arrays.asList("calc.exe"));
        Method m = clazz.getMethod("start");
        m.invoke(o);
    }
}

ProcessBuilder有两个构造函数：

public ProcessBuilder(List command)

public ProcessBuilder(String… command)

用到了第一个形式的构造函数,在 getConstructor 的时候传入的是 List.class
